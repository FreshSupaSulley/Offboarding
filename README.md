# Offboarding
How to get fired from NetJets tutorial

## What
Raises the top speed ceiling from x2 to x15. Any higher and some other part of the webpage breaks.

## Warning
When using this plugin, it is recommended that you set the playback speed to normal before beginning the quiz. There is edge cases where the auto clicker can skip a question entirely resulting in repeated failure.

## Installing
1. `chrome://extensions`
2. Enable *Developer Mode* (top-right corner)
3. Load this repo as an unpacked extension
